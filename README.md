# TL2 Fantasy Game

O **TL2 Fantasy Game** é um aplicativo web onde os usuários podem montar uma equipe de pilotos e construtores inspirada na Fórmula 1.  
Cada participante deve escolher **5 pilotos e 2 construtores** respeitando um orçamento limitado.  
Com base nos resultados reais das etapas, o administrador do sistema lança a pontuação de cada piloto e construtor, e o sistema calcula automaticamente:

- Pontuação da etapa de cada usuário  
- Classificação do GP  
- Pontuação do campeonato  
- Ranking geral dos participantes  

O administrador também pode configurar preços, definir um deadline para alterações de equipe e controlar todo o fluxo das corridas.

O sistema foi desenvolvido em **React + TypeScript** e utiliza **Firebase Firestore** como banco de dados em tempo real.
